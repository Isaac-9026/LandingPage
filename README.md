# ⚽ EA Sports FC 26 — Landing Page

Landing page moderna inspirada en EA Sports FC 26.
Diseñada con un estilo oscuro tipo estadio y efectos visuales sutiles para mejorar la experiencia del usuario.


## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3 (Flexbox, Grid, Variables CSS)
* JavaScript (DOM, eventos)

---

## ✨ Características

* Diseño responsive
* Animaciones suaves (hover y entrada)
* Secciones estructuradas:

  * Hero (inicio)
  * Características
  * Modos de juego
  * Galería
  * Video
  * Descarga
  * Formulario de soporte
* Simulación de envío de formulario con feedback visual
* Footer con redes sociales
